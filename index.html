RLM Skill

Recursive Language Models for Claude Code

RLM treats code as an external database, letting Claude pull infinite context without context rot or bloated prompts.

Explanation

RLM turns your codebase into a live, queryable memory layer. Instead of stuffing everything into the prompt window, it fetches only what it needs, keeping responses sharp even as projects grow.
